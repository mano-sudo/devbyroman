Attendance Monitoring System

A web-based attendance monitoring system built with PHP, MySQL, Bootstrap, and JavaScript. Enables real-time logging of employee time-in/out, automated attendance tracking, and report generation. Features an intuitive admin dashboard for employee and schedule management, with a responsive UI and efficient backend integration for streamlined workforce monitoring.

Project Objective

Build a web-based attendance system for real-time time-in/out tracking, automated reporting, and efficient employee and schedule management through a user-friendly admin dashboard.

Key Features

  • Developed real-time employee time-in/out logging system
  • Created admin dashboard for seamless employee and schedule management
  • Implemented responsive UI using Bootstrap for multi-device access
  • Ensured secure backend integration with PHP/MySQL database

Technologies & Implementation

Built with PHP and MySQL for backend and data management. Used HTML, JavaScript, and Tailwind CSS to create a responsive, user-friendly interface for attendance tracking and admin control.

OutfitHaven: Responsive E-Commerce Platform

A Web-Based E-Commerce Platform for Local Fashion Brands in the Philippines Fashion Depot is a modern web-based e-commerce system built with React, Tailwind CSS, PHP, and MySQL, designed to showcase and support Philippine local fashion brands. The platform offers a smooth and dynamic shopping experience, enabling customers to browse collections in real-time, securely place orders, and engage directly with their favorite local designers.

Project Objective

Build a web-based e-commerce platform using React, Tailwind CSS, PHP, and MySQL for seamless browsing of local Philippine fashion brands, secure checkout, and streamlined product and order management via an intuitive admin dashboard.

Key Features

  • Developed dynamic product catalog with real-time browsing and filtering
  • Implemented responsive, mobile-first UI using Tailwind CSS and React for seamless multi-device shopping
  • Implemented responsive UI using Tailwind CSS for seamless multi-device shopping experience
  • Designed scalable architecture to support multiple local Philippine fashion brands

Technologies & Implementation

Built with PHP and MySQL for backend development and data management. Utilized React and Tailwind CSS to create a responsive, modern, and user-friendly interface that enhances product browsing, shopping experience, and admin control.

PHP ReactJS CSS MySQL Tailwind

Project Objective

Develop a responsive, full-stack burger ordering platform that streamlines the customer experience and provides efficient order and product management for administrators.

Key Features

    Customer Side:

  • Browse interactive menu with product images and categories
  • Add items to cart and update quantity
  • Place orders with real-time total calculation

    • Admin Side:

  • Manage orders product listings, and categories
  • View order history and update order status
  • Responsive admin dashboard with secure login

Technologies & Implementation

Built with PHP and MySQL for backend and data management, using Tailwind CSS and JavaScript for a responsive UI. Includes modular dashboards for users and admins to manage orders and products.
